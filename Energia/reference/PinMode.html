pinMode()

Description

Configures the specified pin to behave either as an input or an output. See the description of digital pins for details.

Syntax

pinMode(pin, mode)

Parameters

pin: the number of the pin whose mode you wish to set

mode: either INPUT or OUTPUT or INPUT_PULLUP

Returns

None

Example


int ledPin = 14;                 // LED connected to digital pin 14

void setup()
{
  pinMode(ledPin, OUTPUT);      // sets the digital pin as output
}

void loop()
{
  digitalWrite(ledPin, HIGH);   // sets the LED on
  delay(1000);                  // waits for a second
  digitalWrite(ledPin, LOW);    // sets the LED off
  delay(1000);                  // waits for a second
}

Note

The analog input pins can be used as digital pins, referred to as A0, A1, etc.
